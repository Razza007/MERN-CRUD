# MERN CRUD

A simple CRUD application built with **Express.js** and **MongoDB** while learning the **MERN stack**.  
This project demonstrates basic **Create, Read, Update, and Delete (CRUD)** operations for managing student data.

## Features
- Add new students
- View all students
- Update student information
- Delete students

## Tech Stack
- Express.js
- MongoDB
- Handlebars (hbs)
- HTML/CSS
